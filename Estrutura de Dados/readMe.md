Esse repositório contém o desenvolvimentos de algumas Estruturas de Dados que aprendi no curso de Algoritmos e Estruturas de Dados no IFMG
e também em um curso da DIO. Sendo assim existe alguns arquivos repetidos, mas é interessante para pode ver a minha evolulção no desenvolvimento
dessas estruturas. Achei muito divertido e interessante o conceito de encadeamento e de ponteiro usado em Java, e a estrutura que senti mais orguhoso
em desenvolver foi a Árvore binária.
